# 🔐 Web Port Scanner

## 📌 Description
Бұл веб-қосымша IP немесе домен бойынша ашық порттарды анықтайды.

## 🚀 Features
- TCP port scanning
- Multithreading (жылдам)
- Web interface (Flask)

## 🛠 Technologies
- Python
- Flask
- Socket

## ▶️ Run
pip install -r requirements.txt  
python app.py  

http://127.0.0.1:5000

## ⚠️ Note
Бұл құрал тек оқу мақсатында.
